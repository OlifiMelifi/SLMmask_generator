# SLM Mask Viewer

Приложение для просмотра 8-битных масок и их FFT (Быстрого Преобразования Фурье).

## Возможности

- **Загрузка масок** из различных форматов:
  - Изображения: PNG, JPG, JPEG, BMP, TIFF
  - NumPy массивы: NPY, NPZ
  
- **Просмотр в 8-битном формате** с автоматической нормализацией
  
- **FFT визуализация** в отдельном окне:
  - Логарифмический или линейный масштаб
  - Отображение амплитудного спектра
  - Статистика FFT (min, max, mean)
  
- **Гистограмма интенсивности** для анализа распределения значений
  
- **Сохранение результатов**:
  - 8-битная маска
  - FFT изображение
  - FFT данные (NumPy формат)

```bash
pip install numpy pillow opencv-python-headless
```

## Использование

### GUI версия

```bash
python mask_viewer.py
```

Интерфейс включает:
1. Кнопка "Загрузить маску" - выбор файла маски
2. Кнопка "Показать FFT" - открытие окна с FFT визуализацией
3. Кнопка "Сохранить 8-bit" - сохранение текущей маски
4. Область просмотра 8-битной маски
5. Гистограмма интенсивности

### CLI версия (для тестирования)

```bash
python test_mask_viewer.py
```

Тестирует основные функции:
- Загрузку маски
- Конвертацию в 8-бит
- Вычисление FFT
- Сохранение результатов

## API основных функций

### `convert_to_8bit(image)`
Конвертирует изображение в 8-битный формат с нормализацией к диапазону [0, 255].

### `compute_fft(image, log_scale=True)`
Вычисляет FFT изображения и возвращает:
- `magnitude` - амплитудный спектр
- `magnitude_display` - спектр для отображения (с логарифмическим масштабом если нужно)

### `load_mask(filepath)`
Загружает маску из файла (изображение или NumPy массив).
