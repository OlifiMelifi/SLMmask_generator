# SLMmask_generator
Тестовый проект для создания алгоритма генерации (оптимизации) 8-битных масок для SLM-модуля на лазерном оборудовании.
Цель данного проекта создать универсальный алгоритм, который будет как создавать 8-битную маску (голограмму), используя метод CGH или итеративный Gerchberg–Saxton (G-S), так и оптимизировать уже существующую маску путем их сложения/вычитания и наложения для получения необходимой формы и плотности пучка.
Ииструменты (фазовые функции), которые задействованы в проекте:
1. Элементы управления направлением (Steering)
Используются, чтобы отделить полезный сигнал от «нулевого порядка» (неотмодулированного света, отраженного от стекла SLM).
1) Линейная решетка - Параллельные полосы (синусоида или пила) - Отклоняет луч под углом. Создает спектр порядков (0, ±1, ±2...).
2) Блаз-решетка - Пилообразный профиль (линейный градиент с обрывом) - Направляет почти 100% энергии в один порядок (обычно +1). На SLM это просто линейный градиент яркости от 0 до 255 с резким сбросом.
3) Дамман-решетка - Структура из прямоугольных зон (бинарная) - Создает массив равномерных точек (спотов) в дальнем поле. Используется для мультипаттернинга.

2. Элементы фокусировки (Focusing)
Используются для сведения пучка в точку или линию в рабочей плоскости.
1) Сферическая линза - Концентрические кольца (зоны Френеля) - Фокусирует пучок в точку. Меняет эффективное фокусное расстояние системы.
2) Цилиндрическая линза - Полосы, меняющие частоту по одной оси - Фокусирует только по оси X. Превращает точку в линию. Ключевой элемент для прямоугольника.
3) Астигматическая линза - Эллиптические кольца - Разные фокусные расстояния по X и Y. Используется для коррекции эллиптичности пучка лазера.

3. Элементы формирования профиля (Beam Shaping)
Преобразует Гауссов профиль (колокол) в нужный (прямоугольник, кольцо, плоская шляпа).
1) Асферическая фаза - Плавные искажения колец линзы - Превращает Гаусс в Top-Hat (равномерное пятно). Рассчитывается через метод Ray Mapping.
2) Вихревая фаза - «Спираль» или «штопор» (разрыв фазы) - Создает пучок с орбитальным моментом (кольцо с темной серединой). Для микроскопии, манипуляций.
3) Аксикон - Концентрические кольца с линейным шагом - Создает Бесселев пучок (не дифрагирующая точка/линия на длинном расстоянии). Для резки внутри стекла.
4) Кубическая фаза - Искаженная решетка - Используется для увеличения глубины резкости (Wavefront Coding).

4. Элементы коррекции (Correction)
Компенсируют недостатки оптики и самого SLM.
1) Полиномы Зернике - Различные симметричные узоры - Коррекция аберраций системы (кома, астигматизм, сферическая аберрация).
2) Коррекция плоскостности - Медленный градиент - Карта измеренных искажений - SLM часто имеет кривое зеркало. Загружается статическая маска-компенсатор.
3) Gamma / LUT коррекция - Не видно (внутри кода) - Таблица соответствия - Компенсация нелинейности ЖК (чтобы 128 было ровно π).
